It is a Expense-tracker application which allows user to record their daily expenses.
It solves a real-world problem like Personal Expense Management.

//Features-
1.Add daily expenses with date and amount.
2.Menu-based category selection- Categories:-
                                 i.Food
                                 ii.Travel
                                 iii.Education
                                 iv.Entertainment
                                 v.Bills
                                 vi.Others
3.View all recorded expenses.
4.File-based data persistence
5.User-friendly interface

//Concepts Used:-
1.Structures- for storing expense details such as date,amount and category.
2.File Handling- for saving and retrieving expense data using text files.
3.do-while Loop- to keep application running until user exits.
4.Switch statements- to handle menu options and map category numbers to names.
5.if-else
6.Functions
